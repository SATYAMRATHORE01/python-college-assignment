Python Print and Conversion Functions

Printing Function (print())

The print() function in Python is a fundamental tool for displaying information in the console or terminal. It is versatile and allows for various formatting options. Key aspects of the print() function include:

  • Basic Usage: The simplest usage involves passing a string or variable to print().
    • ```python print("Hello, World!") ```
  • Multiple Arguments: Multiple items can be printed by separating them with commas.
    • ```python name = "Alice" age = 30 print("Name:", name, "Age:", age) ```
  • Formatted Output: You can format the output using f-strings or the format() method.
    • ```python name = "Bob" age = 25 print(f"Hello, {name}! You are {age} years old.") ```
  • End and Sep Parameters: The end parameter specifies what to print at the end (default is newline \n), and sep specifies the separator between arguments (default is space ' ').
    • ```python print("This", "is", "a", "sentence", end="!!\n", sep="-") ```
  • File Parameter: You can redirect the output to a file by specifying the file parameter.
    • ```python with open("output.txt", "w") as file: print("Hello, file!", file=file) ```

Conversion Functions

Conversion functions in Python are used to change the data type of a value or variable. Here are some common conversion functions:

  • str(): Converts a value to a string.
    • ```python x = 42 str_x = str(x) # Result: '42' ```
  • int(): Converts a value to an integer.
    • ```python y = "10" int_y = int(y) # Result: 10 ```
  • float(): Converts a value to a floating-point number.
    • ```python z = "3.14" float_z = float(z) # Result: 3.14 ```
  • bool(): Converts a value to a boolean.
    • ```python truthy_value = 42 bool_truthy = bool(truthy_value) # Result: True ```
