String Operators in Python

Strings in Python support various operators for common operations like concatenation, repetition, and comparison. Here are some key string operators:

  1. Concatenation (+): The + operator is used to concatenate (combine) two strings. 
    str1 = "Hello"
str2 = " World"
result = str1 + str2  # Result: "Hello World"
  2. Repetition (*): The * operator is used for string repetition. 
    original = "Python"
repeated = original * 3  # Result: "PythonPythonPython"
  3. Membership (in and not in): The in and not in operators are used to check if a substring is present or not in a given string. 
    text = "Python is powerful"
check1 = "Python" in text  # True
check2 = "Java" not in text  # True
  4. Comparison (==, !=, <, >, <=, >=): Strings can be compared lexicographically. 
    str1 = "apple"
str2 = "banana"
result = str1 < str2  # True (compared based on lexicographical order)
  5. Indexing ([]): Strings support indexing to access individual characters. 
    text = "Python"
first_char = text[0]  # Result: "P"
  6. Slicing ([:]): Slicing is used to extract a portion of a string. 
    text = "Python"
sliced_text = text[1:4]  # Result: "yth"
  7. String Formatting (% and str.format()): Old-style string formatting using % and newer-style using str.format() are used to insert values into a string. 
    name = "Alice"
age = 30
message = "Hello, my name is %s and I am %d years old." % (name, age)

    Using str.format():

    message = "Hello, my name is {} and I am {} years old.".format(name, age)
  8. String Methods: Various string methods like upper(), lower(), strip(), replace(), etc., are used for string manipulation. 
    text = " Python is fun "
uppercase_text = text.upper()  # Result: " PYTHON IS FUN "
stripped_text = text.strip()   # Result: "Python is fun"

Numeric Data Types in Python

Python supports several numeric data types for working with numerical values. Here are the main numeric data types:

  1. Integers (int): Integers represent whole numbers, positive or negative, without any decimal points. 
    x = 5
y = -10
  2. Floating-Point Numbers (float): Floating-point numbers represent real numbers and include a decimal point. 
    pi = 3.14
height = 5.9
  3. Complex Numbers (complex): Complex numbers consist of a real and an imaginary part. 
    z = 3 + 2j
  4. Numeric Operations: Python supports various numeric operations, including addition (+), subtraction (-), multiplication (*), division (/), exponentiation (**), and modulo (%). 
    result = 10 + 5    # Addition
result = 15 - 3    # Subtraction
result = 2 * 4     # Multiplication
result = 8 / 2     # Division
result = 3 ** 2    # Exponentiation
result = 10 % 3    # Modulo (remainder after division)
  5. Numeric Type Conversion: You can convert between numeric types using functions like int(), float(), and complex(). 
    x = 10
y = float(x)      # Convert to float
z = complex(x)    # Convert to complex

Understanding these string operators and numeric data types is essential for effective programming in Python, whether you are working with text or numerical values.
